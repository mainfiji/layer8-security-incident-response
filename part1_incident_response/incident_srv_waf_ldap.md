### Выявленный путь атаки
1. **Проникновение:** Через уязвимость в веб-приложении на srv_waf (SQL-инъекция)
2. **Повышение привилегий:** Использование локальной уязвимости для получения root-доступа
3. **Установка бэкдора:** Создание скрытого пользователя backdoor
4. **Горизонтальное перемещение:** Использование скомпрометированных учётных данных администратора LDAP для доступа к srv_ldap

### Оценка
- **Тип:** Полная компрометация критичных серверов
- **Критичность:** Критическая
- **Статус:** Активная атака

На межсетевом экране fw добавлены правила:
```bash

Результат: Серверы полностью изолированы, кроме управляющих соединений с админских хостов.

Серверы выведены из эксплуатации
