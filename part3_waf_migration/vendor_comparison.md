# Сравнительный анализ WAF-решений

## Критерии сравнения

1. Графический интерфейс
2. Визуализация и агрегация логов
3. Анализ трафика (семантический / лексический)
4. Производительность и задержки
5. Стоимость / лицензия
6. Защита от ботов
7. GeoIP блокировка
8. Работа в Docker
9. Интеграция с Wazuh

---

## Таблица сравнения

| Критерий | BunkerWeb | SafeLine | ModSecurity (старый) |
|----------|-----------|----------|----------------------|
| **Графический интерфейс (GUI)** | Есть (веб-интерфейс для мониторинга, но не для всех функций) | Полноценный веб-интерфейс с дашбордами и управлением | Отсутствует (только текстовые логи и конфиги) |
| **Визуализация и агрегация логов** | Базовая визуализация, требуется интеграция с SIEM | Развитая система агрегации, группировка инцидентов | Отсутствует, требуется внешний SIEM |
| **Анализ трафика** | Комбинация сигнатур и поведенческого анализа | Собственный движок SAW (Semantic Analysis for Web), контекстный анализ | Основан на регулярных выражениях (OWASP CRS), высокий процент ложных срабатываний |
| **Производительность и задержки** | Средняя, зависит от количества правил | Низкая задержка, высокая пропускная способность | Высокая нагрузка на CPU, особенно при большом количестве правил |
| **Бесплатность / Open Source** | Есть бесплатная версия с ограничениями (по правилам), полностью открытый код | Есть бесплатная версия (Community Edition) с ограничением по сайтам (до 10) | Бесплатный, open source |
| **Защита от ботов** | Базовые механизмы (CAPTCHA, rate limiting) | Встроенная защита (анализ поведения, User-Agent, TLS fingerprint) | Отсутствует (требуются дополнительные модули) |
| **GeoIP блокировка** | Поддерживается (через базы GeoIP) | Поддерживается (настраивается в GUI) | Поддерживается через внешние модули |
| **Работа в Docker** | Нативная поддержка (официальный образ) | Официальная поддержка Docker, разворачивается одной командой | Требуется ручная настройка связки Nginx+ModSecurity в контейнере |
| **Интеграция с Wazuh** | Логи в syslog, есть плагины | Логи в syslog, формат JSON, легко парсится | Логи доступны, но сложны для парсинга из-за большого объёма |

## Итоговая оценка

### BunkerWeb
**Плюсы:**
- Полностью бесплатный и open source
- Есть GUI для мониторинга
- Поведенческий анализ снижает ложные срабатывания
- Встроенная защита от ботов
- Поддержка Docker и интеграции с Wazuh
- Хорошая документация

**Минусы:**
- GUI не полностью управляющий (часть настроек через переменные окружения)

**Рейтинг:** ⭐⭐⭐⭐☆

### SafeLine
**Плюсы:**
- Отличный GUI (дашборды, управление)
- Семантический анализ (SAW) — очень низкая нагрузка
- Эффективная защита от ботов
- Docker-развёртывание

**Минусы:**
- Бесплатная версия с ограничениями (10 сайтов)
- Меньше документации, чем у BunkerWeb

**Рейтинг:** ⭐⭐⭐⭐☆

### ModSecurity (старый)
**Плюсы:**
- Бесплатный
- Гибкая настройка

**Минусы:**
- Нет GUI
- Высокая нагрузка
- Много ложных срабатываний
- Сложная интеграция с современными SIEM

**Рейтинг:** ⭐⭐☆☆☆

## Рекомендация

Выбрано решение **BunkerWeb**, так как оно полностью удовлетворяет ключевым требованиям:
- GUI с визуализацией
- Бесплатность (бюджет ограничен)
- Семантический анализ (хотя и не заявлен, но поведенческий анализ есть)
- Защита от ботов и GeoIP
- Docker и интеграция с Wazuh
- Лучшая документация по сравнению с SafeLine

SafeLine — достойная альтернатива, но платные ограничения и менее развитая документация склоняют чашу весов в пользу BunkerWeb.
